# 旅算家 · 多Agent智能旅游行程规划系统

> **5 分钟完成原本 8 小时的行程规划** — 输入预算和偏好，6 个 AI 智能体协作完成目的地推荐、交通比价、住宿匹配、行程安排，全程预算闭环。

---

## 预览

直接打开 `index.html` 即可体验，无需安装任何依赖。

| 引导页 | AI 规划首页 | 推荐结果页 | 活动详情页 |
|:------:|:-----------:|:----------:|:----------:|
| 品牌展示 → 进入 | 输入预算/偏好 | 预算条 + 交通/住宿 | 地图 + 时光轴 |

## 核心功能

### 🤖 6 Agent 智能协作
| Agent | 职责 |
|-------|------|
| **PreferenceAgent** | 分析用户偏好，补充兴趣标签 |
| **DestinationAgent** | 基于预算和地域推荐 3 个目的地 |
| **TransportAgent** | 搜索去程/返程交通（航班/高铁） |
| **HotelAgent** | 匹配合适的住宿选项 |
| **ActivityAgent** | 规划每日行程活动 |
| **BudgetAgent** | 核算总花费，超预算自动循环调整 |

### 📱 交互特性
- 目的地横向滚动选择，勾选标识已选
- 交通/住宿/活动三段切舱
- 多日行程时光轴
- Leaflet 交互地图（全屏展开）
- 预算进度条实时反馈
- 切换目的地时所有数据联动更新

### 🎨 设计风格
轻氧薄荷奶油风 — 薄荷绿主色、奶油白卡片、大圆角、柔和阴影

## 技术栈

| 层 | 技术 | 选型理由 |
|------|------|---------|
| **前端 UI** | 原生 HTML/CSS/JS + Feather Icons | 轻量无依赖，移动端响应式 |
| **地图** | Leaflet + OpenStreetMap | 免费开源，无需 API Key |
| **后端框架** | Python FastAPI + Uvicorn | 高性能异步框架，原生支持 Pydantic |
| **LLM 引擎** | 通义千问 (DashScope API) | 国内可用，兼容 OpenAI 接口格式 |
| **LLM 调用** | HTTPX (异步) | 非阻塞请求，适合并行调用场景 |
| **数据模型** | Pydantic v2 | 类型安全，全局状态流转 |
| **异步调度** | asyncio + asyncio.gather | Agent 并行执行 |
| **配置管理** | pydantic-settings + .env | 环境变量管理，多环境切换 |
| **日志** | Loguru | 结构化日志，自动轮转 |
| **Mock 数据** | 内置 Mock 数据源 | 零成本演示，可替换为真实 API |
| **前端构建** | 无构建工具 (原生) | 零配置，开箱即用 |

###  Agent 职责定义

| Agent | 输入 | 输出 | 决策逻辑 | 失败影响 |
|-------|------|------|---------|---------|
| Preference | 用户原始输入 | 补充后的偏好（含默认兴趣标签） | 规则引擎 | 全流程终止 |
| Destination | 偏好 | Top3目的地+推荐理由 | 多维度加权评分 | 全流程终止 |
| Flight | 出发城市+目的地+日期 | 航班列表+推荐 | 价格50%×时长30%×中转20% | 不影响其他并行Agent |
| Hotel | 目的地+日期+风格 | 酒店列表+推荐 | 风格匹配+距离评分 | 不影响其他并行Agent |
| Activity | 目的地+天数+兴趣 | 每日行程 | 兴趣匹配+时间槽分配 | 不影响其他并行Agent |
| Budget | 所有费用汇总 | 预算明细+调整建议 | 费用汇总+渐进式降级策略 | 使用上次可用结果 |
